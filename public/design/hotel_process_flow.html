1

3
 검색 사용자 도시·날짜·인원 입력 후 검색 검색 조건 파싱 및 쿼리 실행 properties + room_availability JOIN 지도 API (좌표 변환)
4

5
 상세 사용자 숙소 카드 클릭 → 상세 페이지 숙소 정보 + 가용 객실 조회 properties, room_types, reviews 조회 -
6
 선택 사용자 날짜·객실 타입 선택 가용 재고 실시간 확인 room_availability 조회 (날짜 범위) -

9
 재고잠금 시스템 결제 페이지 진입 Redis 임시 재고 홀드 (10분) - Redis SETNX

 결제 사용자 결제 수단 선택 및 결제 진행 PG 결제 요청 생성 payments 레코드 생성 (pending) 토스페이먼츠 / PG사
